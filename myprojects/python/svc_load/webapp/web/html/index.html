驾驶舱

  • 压测开始前的主控页面，包括压测数据构造、节点操作、节点数据监控等

停机坪

  • 压测负载机管理页面，可对负载机进行增删改查操作，只有状态为启用的slave机会被用于压测

Locust

  • Locust压测管理监控页面，可启动/停止压测，展示汇总后的实时压测数据

压测记录

  • 记录每次压测的信息，包括并发数、RPS、图表曲线等，图表数据默认保存30天

Stats Tab Content
启动Master

  • 启动Master节点，默认已启动。访问Locust页面，如无法打开，则通过节点操作-重启Master

启动Slave

  • 节点操作-重启slave：会启动停机坪中状态为启用的节点，压测节点数量为状态是启用的slave节点数总和，可通过驾驶舱页面的节点监控查看各slave节点的状态，初始状态Connected

构造压测数据

  • 当Master和Slave已启动并且状态为connected，则可以开始构造压测数据；如需重新构造数据 则需要先重置数据，重置操作会初始化Master和Slave的状态，并清空各节点的数据

开始压测

  • 压测数据构造完成后，即可点击开始压测按钮或者左侧导航Locust跳转到压测界面开始测试
